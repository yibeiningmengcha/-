# 中国动画电影市场全景洞察大屏 - 环境与技术栈
# Environment & Tech Stack

===============================================
  一、运行环境 (Runtime Environment)
===============================================

Node.js: >= 18.0.0 (推荐 LTS v20.x / v22.x)
npm: >= 9.0.0 (随 Node.js 安装)
操作系统: Windows / macOS / Linux
浏览器: Chrome / Edge / Firefox (现代浏览器)

===============================================
  五、启动命令 (Quick Start)
===============================================

# 1. 进入项目目录和安装依赖 (首次运行)
cd dashboard-frontend    # 进入项目目录
# 安装依赖
npm install

# 2. 启动开发服务器
npm run dev
# 访问 http://localhost:5173

# 3. 生产构建
npm run build

# 4. 预览构建结果
npm run preview
